# Aizen REST API Docs

![Aizen Logo](./src/picture2.png)

Aizen REST API Docs adalah **web interface interaktif** untuk menampilkan dokumentasi REST API. Memiliki fitur **search endpoint**, **execute API langsung**, **theme switcher**, dan **background audio**.

---

## Fitur

- Menampilkan **kategori dan endpoint API** secara dinamis dari file `endpoints.json`.
- **Execute API** langsung dari web interface.
- **Refresh/reset** parameter dan result.
- **Search endpoint** real-time.
- **Dark/Light theme toggle** 🌙 / ☀️.
- **Audio toggle** 🔊 / 🔇.
- **Validasi parameter wajib**: jika belum diisi, akan muncul notifikasi.
